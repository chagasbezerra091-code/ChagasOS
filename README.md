# ChagasOS – Android Custom ROM

ChagasOS é uma ROM personalizada baseada no Android Open Source Project (AOSP),
com modificações próprias e suporte inicial para o dispositivo **LG K41S (LM-K410)**.

Este repositório contém o **código-fonte principal** do sistema operacional:
- Frameworks
- System
- Hardware
- Kernel (caso incluído)
- Drivers HAL
- Bibliotecas nativas
- Ferramentas de build
- Packages
- Vendor blobs (se aplicável)
